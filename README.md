# Budget Tracker

A Python tool that analyzes monthly spending and earnings, 
calculates net surplus or deficit per month, and visualizes 
the results with matplotlib charts.

## Built with
- Python
- Pandas
- Matplotlib
- Jupyter Notebooks

## What it does
- Tracks spending and earnings by month
- Automatically labels each month as surplus or deficit
- Generates two charts: spending vs earnings, and net by month
